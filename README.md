# tic-tac-toe-cli  

# To install
1. Clone this repository
````
git clone https://github.com/ShiniDev/tic-tac-toe-cli.git  
cd tic-tac-toe-cli/
````
2. Clean the build directory if it exists  
In Linux  
````
rm -rf build/
````
In Windows  
````
rmdir /Q /S build
````
3. Run cmake
````
cmake -S . -B build/
````
4. Run make
````
make -C build/
````
5. Run main
````
./build/src/main
````
